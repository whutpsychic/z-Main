
新增视觉组件:
1.修改db.js(新增菜单及相关参数)
2. npm run pages（此命令之后，components文件夹内即已新增了新的组件文件夹）



前端代码工程化解释：

1.npm run pages:
自动化配置左侧菜单、路由和右侧UI视图匹配（每次新增组件时运行此命令）
---------
加载所有源自“src/components/”的文件夹，将之引入并把相关信息写到“src/pages.js”当中,包括命名检测等，以便于自动加载路由等操作。
